# Food2Good (Web UI Simulation)

A project submission for the **Assessment of Learning (AOL)** - Software Engineering Course.

Food2Good is a frontend UI/UX simulation of a surplus food marketplace. The project demonstrates a basic web application workflow, from browsing available deals to a dynamic checkout process.

## ‚ú® Key Features
- **Food Catalog**: A responsive display of surplus food items available for purchase.
- **Dynamic Cart Logic**: Automated subtotal and delivery fee calculations based on user input.
- **Merchant Dashboard**: UI mockups for sales statistics and surplus item management.
- **Form Validation**: Client-side validation for the checkout process (Name and Address fields).
- **Responsive Design**: Mobile-friendly layouts built with CSS Flexbox and Grid.
